# ⚡ Lección 4: Entradas Analógicas y Divisor de Voltaje en Tinkercad

Hasta ahora, nuestro Arduino solo ha enviado señales de salida (ALTO o BAJO). En esta lección aprenderemos cómo recibir señales del exterior usando las **Entradas Analógicas** (pines **A0** a **A5**). 

Para comprender cómo mide voltajes el Arduino, primero construiremos un circuito simulado llamado **Divisor de Voltaje** utilizando resistencias fijas y un multímetro virtual en Tinkercad.

---

## 💻 Conceptos Clave
1. **Entradas Analógicas:** A diferencia de las digitales (que leen solo 0V o 5V), las analógicas miden valores intermedios de voltaje (ej. 2.5V). El Arduino traduce ese voltaje a un número entre **0 y 1023** usando su convertidor analógico-digital (ADC).
2. **Divisor de Voltaje:** Es un circuito en serie que divide el voltaje de entrada (5V) en fracciones proporcionales según los valores de las resistencias utilizadas.

### Pasos para realizar el montaje:
1. Coloca un **Arduino Uno R3** y una **Placa de pruebas (protoboard)**.
2. Conecta **5V** y **GND** de Arduino a los buses positivo y negativo de la protoboard.
3. Coloca **dos resistencias de 10k ohmios** en serie (colores: marrón, negro, naranja). Una patita de la primera resistencia va a 5V; una patita de la segunda va a GND. Las otras dos patitas se unen en la misma columna.
4. Conecta el pin analógico **A0** de Arduino a esa columna de unión intermedia.
5. Arrastra un **Multímetro** virtual, ponlo en modo **Tensión (V)**, y conecta su terminal positivo (+) a la columna intermedia, y su terminal negativo (-) a GND.

---

## 🔬 Simulando y Midiendo el Voltaje

1. Haz clic en **Iniciar Simulación**.
2. El multímetro debería marcar exactamente **2.50 V**. ¿Por qué? Como ambas resistencias son idénticas (10kΩ cada una), el voltaje de 5V se divide equitativamente a la mitad.
3. Haz clic en **Código**, selecciona **Texto** y sube este programa para ver qué lee el Arduino:

```cpp
void setup() {
  Serial.begin(9600); // Iniciamos el puerto serial
}

void loop() {
  int lectura = analogRead(A0); // Leemos el voltaje (0 a 5V) como número (0 a 1023)
  Serial.println(lectura);      // Imprimimos el número
  delay(200);
}
```
4. Abre el **Monitor en Serie**. Verás que imprime un número muy cercano a **512** (la mitad exacta de 1023).
