## 💻 Programación Lógica (SI / SINO)

En el panel de **Código** en Tinkercad, utilizaremos un bloque condicional **si / si no** para controlar nuestro LED:

1. **La Condición:** Si la lectura analógica en A0 es menor que nuestro **Umbral** (por ejemplo, `400`), significa que está oscuro.
2. **Acción 1 (SÍ):** Encendemos el LED (Pin 5 en ALTO).
3. **Acción 2 (NO):** Apagamos el LED (Pin 5 en BAJO).

### Código C++ equivalente de Tinkercad:
```cpp
void setup() {
  pinMode(5, OUTPUT);  // Pin del LED como salida
  Serial.begin(9600);  // Monitor serial
}

void loop() {
  int luz = analogRead(A0); // Leemos el sensor
  
  if (luz < 400) {          // Condición de oscuridad
    digitalWrite(5, HIGH);  // Encendemos el LED
  } else {
    digitalWrite(5, LOW);   // Apagamos el LED
  }
  delay(100);
}
```
